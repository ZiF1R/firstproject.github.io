Введение
JavaScript был создан в 1995 году как инструмент, предоставляющий вебдизайнерам возможности программирования. Все современные браузеры имеют поддержку JavaScript. JavaScript встраивается прямо в веб-страницы и исполняется браузером во время их загрузки. JavaScript добавляется на веб-страницы с помощью тэга:
Html 
<script>...</script>
В следующем примере на страницу выводится текст «Я изучаю JavaScript».
Html 
<script type="text/javascript">
    document.write("Я изучаю JavaScript.");
</script>
JavaScript код также может храниться во внешнем текстовом файле с расширением .js и подключаться к страницам следующим образом:
Html 
<script type="text/javascript" src="ex.js"></script>
