Лабораторная работа 1
Знакомство с HTML5. Элементы video и audio. Геолокация.
Цель работы
Научиться подключать на веб-страницу аудио и видео файлы, определять местоположение пользователя.
Задания
Задание 1
Создать новую веб-страницу. Вставит заголовок в центре страницы «Я, ФИО, изучаю HTML5». Подсветить ФИО с помощью тега <mark>.
Решение
HTML5 
Я, <mark>Добриян Александр Витальевич</mark>, изучаю HTML5.
Я, Добриян Александр Витальевич, изучаю HTML5.
Задание 2
Вставить аудио и видео файлы.
Решение
HTML5 
<video src="ocean.mov" width="50%" height="100%" controls></video>
<audio src="sound.mp3" controls></audio>
Задание 3
Определить свои координаты с помощью геолокации.
Решение
HTML5 
<button onclick ="getLocation()">Вычислить координаты</button>
Java Script 
function getLocation() {
    navigator.geolocation.getCurrentPosition(showCoordinates);
}

function showCoordinates(position) {
    document.write("Широта: " + position.coords.latitude + "<br>")
    document.write("Долгота: " + position.coords.longitude)
}
